# Interactive Demo Site

This folder contains a static interactive demo page for external viewers.

## Includes

- Photo Log demo wizard simulation
- Intelligent QI demo check simulation

## GitHub Pages (recommended)

In the target repository:
1. Go to **Settings -> Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / folder: `/collected_projects/site` (or copy this into `/docs`)
4. Save and open the published URL

No backend is required; this is static HTML/CSS/JS and portfolio-safe.
