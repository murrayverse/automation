## How To Browse

1. Open a project `README.md` for the overview.
2. Open its `demo_page.md` for a GitHub-friendly “webpage style” walkthrough.
3. Open the `sample_*.py` file to review curated implementation snippets.

## Why This Structure

- Keeps production modules untouched.
- Makes sharing specific capabilities easier with stakeholders.
- Creates a clear bridge from business feature -> technical implementation.
