# Intelligent QI - Demo Page

## Problem It Solves

Manufacturing and supplier quality issues often require:
- structured checks,
- clear NC severity,
- ownership/liability assignment,
- and shareable outbound reports.

## Core Flow

1. Create or receive a QI check tied to a plan/document.
2. Run checklist statuses (`pending/pass/fail/na`) with notes.
3. Flag NC as major/minor.
4. Generate SCAR output for external sharing.
5. Keep traceability via linked attachments and chatter.

## Highlighted Capabilities

- **Checklist integrity**: pass only when mandatory steps are completed.
- **NC classification**: explicit major/minor severity.
- **Liability tracking**: Makerverse / Customer / Supplier.
- **Complaint traceability**: customer complaint flag.
- **Export workflow**: SCAR as XLSX.
